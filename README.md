# ЗдоровоЕда — лендинг сервиса доставки здорового питания

> **Учебный проект** к практическому занятию 7: *Работа с CSS-фреймворками*  
> Тема: одностраничный минималистичный лендинг для сервиса доставки сбалансированных рационов.


## Описание проекта

Этот проект — каркас будущего веб-приложения **«ЗдоровоЕда»**, ориентированного на целевую аудиторию: людей, заботящихся о здоровье, спортсменов и тех, кто хочет перейти на осознанное питание без лишних усилий.


## Структура сайта

Сайт состоит из следующих секций:

**Шапка** Логотип *«ЗдоровоЕда»*, навигация (якорные ссылки), кнопка «Заказать»
**О еас** Краткое описание философии сервиса
**Преимущества** 4 карточки: *Свежие ингредиенты*, *Персональный план*, *Без ГМО*, *Доставка за 30 мин*
**Отзывы** 3 отзыва с именами и возрастом
**Футер** Контакты (телефон, email, адрес), соцсети (VK, Telegram)

> Карта сайта в формате Mind Map создана в BoardMix и приложена как `site_mindmap.png`.


## Технологии

**HTML5** Семантическая разметка (`<header>`, `<main>`, `<section>`, `<footer>`)
**Bulma** CSS-фреймворк на Flexbox — обеспечивает адаптивность, чистоту кода и современный минималистичный дизайн
**Git** Контроль версий и публикация через GitHub Pages
